Strategy Launchpad API

Run script

Description :  run script
Form Parameter: script : script that will be executed. 
Cookies Parameter: ALSID : session ID, ALUSER : active user.
Example : https://dev-qlib.alphien.com/scheduler/api/action/launchpad/run/script
Method: POST
Returns:
	
	{
	  "result": "Accepted and will be processed.Script ID : 1225"
	}

Cancel script

Description :  cancel script execution
Form Parameter: id : id of specific script that is being executed.
Cookies Parameter: ALSID : session ID, ALUSER : active user.
Example : https://dev-qlib.alphien.com/scheduler/api/action/launchpad/cancel/script
Method: POST
Returns:
	
	{
	  "result": "Rscript execution cancelled"
	}

Show status

Description :  get status of specific script
Path Parameter: id : id of specific script.
Cookies Parameter: ALSID : session ID, ALUSER : active user.
Example : https://dev-qlib.alphien.com/scheduler/api/action/launchpad/show/status/id/1227
Method: GET
Returns:
	
	{
	  "result": "Current Status : Executing"
	}

Get console message

Description :  get console message of executed script
Path Parameter: id : id of specific script.
Cookies Parameter: ALSID : session ID, ALUSER : active user.
Example : https://dev-qlib.alphien.com/scheduler/api/action/launchpad/show/console/id/1227
Method: GET
Returns:
	
	{
	  "result": "{\"consoleoutput\":\"[1] \\\"test\\\"\\n\"}"}

Get backtest result

Description :  get backtest result of executed script. Result can be *.html or *.png file. 
Path Parameter: id : id of specific script.
Cookies Parameter: ALSID : session ID, ALUSER : active user.
Example : https://dev-qlib.alphien.com/scheduler/api/action/launchpad/show/backtest/id/1227
Returns:
	
	{
	  "result": "/backtest/61c570a8596852b36b56b083a9f0b213dd867f0b0b9bdce482a3c460b9b0d6b9.html"
	}
